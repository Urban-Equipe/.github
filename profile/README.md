### About Urban Equipe

Urban Equipe is an association that is committed to the democratization of urban development and design. Based in ZÃ¼rich Switzerland, we are organized as a non-profit association and as such we are committed to our purpose in all our projects. To achieve our purpose, we initiate participatory projects, organizes knowledge exchange and carry out assignments for organizations, with a focus on municipalities and civil society actors in the field of urban development.

Urban Equipe is an official [Decidim](https://github.com/decidim/decidim) partner.

### What we do on Github

This GitHub space is used by Urban Equipe to:

- document and maintain digital tools used in participatory urban development processes
- host customizations and extensions related to [Decidim](https://github.com/decidim/decidim)
- share open-source code and resources from Urban Equipe projects
- collaborate with municipalities, civil-society partners, and other accomplices

If you are looking to learn more about us, visit: `https://www.urban-equipe.ch`

### Repository structure

Typical repositories in our organization include:

- **Tools and scripts**: utilities that support participatory processes (e.g. data preparation, exports, reporting)
- **Documentation**: process documentation, guides, and templates for participatory urban development


### Contributing

Contributions that improve the tools, documentation, and processes around democratic urban development are welcome.

- **Issues**: Use GitHub Issues to report bugs, propose improvements, or suggest new features.
- **Pull requests**: Fork the relevant repository, create a feature branch, and open a pull request with a clear description in English, German, or French where possible.
- **Discussions**: When enabled, use Discussions (or Issues) to talk about ideas before implementing larger changes.

Please respect the participatory values of Urban Equipe: transparency, inclusion, and openness to diverse perspectives.
